学习折腾出的小实例 demo

编程写代码是一门艺术，特别是对于前端。简洁、美观和新颖，是我不懈的追求。

实例1. 随机名言获取器

使用 jQuery、Bootstrap，以及 Vue 的数据绑定功能，实现:

1.点击一个按钮就生成一个新的随机引用短语
2.点击一个按钮就把这个引用短语发到 Facebook/Twitter/Weibo 中

  • demo 地址：View on CodePen
  • 源码：View on GitHub

实例2. 维基百科查看器

该Demo 使用 jQuery，通过 AJAX 方式向维基百科发送 POST 请求，在不刷新页面的情况下随着用户输入字符，实时改变页面布局，将查询到的结果以 词条名+摘要 的形式显示在查询结果列表中，当用户点击后即跳转到维基百科对应词条页面。用户还可以通过点击按钮，查看随机词条

  • demo 地址：View on CodePen

  • 源码：View on GitHub

实例3. 本地天气预报

该 Demo 使用和风天气 API ，通过解析 JSON 获取相应内容，实现根据 IP 查询当前用户所在城市以及城市当前天气情况和未来七天天气预报
